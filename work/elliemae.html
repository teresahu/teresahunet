EllieMae Encompass NextGen Prototype

Summary

Understand how mortgage loan officers use Encompass throughout the loan process, identify pain points, propose new designs, and discover which is design the best for users.

Team: me & a principal designer
Collaborated with: engineers, PMs, designers, researchers, users
Timeline: 20 days

interaction design prototyping CSS3 animations

I joined this project in its final phase of the design process: prototyping. We started off with knowledge transfer, a deep dive into the world of mortgage loan officers. The principal designer shared his insights and mockups with me, and together we came up with ways to enhance the overall user experience of mortgage loan application software.

The mortgage loan application process requires filling out pages and pages of forms. Based on previous user research, loan officer productivity was impeded by to confusing navigation and long scrolling pages. To address this, we prioritized (1) making the navigation flow clear and (2) helping users stay focused on form pages.

Animations can be really helpful affordances for users to stay focused. In a few hours, I created this demo to show an example of this.


Another example of how animation and color can help the mortgage loan officer keep track of where he or she is on the page. The form field expands in size and has a focus state outline when active.


The most challenging problem I had to solve throughout this project: how will the loan officer know when his input is part of a larger system and it affects the overall mortgage application?

Some sort of indication would be necessary. Keeping in mind our goal was to help the user focus, a slight signal would be ideal. Enough to be noticed in one’s peripheral vision, but not so much movement that it draws full attention.

The appropriate application of a subtle animation can signal change without distracting user from their task at hand. Inspired by stock ticker prices changing, I created light blue highlights that animate whenever the loan officers enters input that affects the mortgage application. Check it out below.


Because my deliverables were created in HTML, CSS, and JavaScript, EllieMae’s engineering team will be able to use the animation code directly in the final product. The interactive prototype is functional and ready for user testing.
